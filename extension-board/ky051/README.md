This is based on [this guide](http://sensorkit.joy-it.net/index.php?title=KY-053_Analog_Digital_Converter).

Use the board in *5V*.

The connections should be (`ky051: board`):
* VCCa 	:	output 3,3V (use it where you need)
* VCCb 	: 	V (input 5V)
* GND 	: 	Gput
* A1-A4	: 	3,3V output
* B1-B4	:	5V output
